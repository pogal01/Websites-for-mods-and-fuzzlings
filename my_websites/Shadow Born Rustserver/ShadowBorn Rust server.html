What is this server about ?

Shadow Born is a server for the game Rust.
The reason my friend and I created the server was because we realy wanted a server that was a bit like GDI a server I used to play on legacy ( the old version of rust).

The server will be 3x gather faster crafting and you can stack items x2 and resorces (wood,stone,metal,sulfur,charcol,lowgrade,animalfat,cloth and leather) will be x3stacks.
